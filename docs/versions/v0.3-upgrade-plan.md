# Upgrade Plan: v0.2 to v0.3

## Objective
Transform the PAAX AI platform from a Vite+Express demo (v0.2) into a production-ready, scalable, multi-tenant workspace architecture (v0.3).

## What is Changing?

### 1. Repository Structure
- **Migration to Monorepo**: Adopting a monorepo structure (pnpm/Python) to manage frontend, backend, and AI services in one place, ensuring schema consistency.

### 2. Frontend Modernization
- **Next.js App Router**: Moving from Vite to Next.js for better routing, server-side rendering, and SEO/performance.
- **Project-Centric Navigation**: The UI is shifting from flat tabs to a full dashboard (`/dashboard/[projectId]/...`) with 12 distinct sidebar modules.

### 3. Backend Overhaul
- **FastAPI Core Engine**: Replacing Express.js with Python/FastAPI. Python is chosen for its superior ecosystem for data processing, pandas, and deterministic calculations.
- **Genkit AI Orchestrator**: Replacing raw Gemini API calls with Firebase Genkit for robust tool calling, routing, and observability.

### 4. Database & Storage
- **Firestore Integration**: Moving from static JSON to a live NoSQL database with proper collections (`users`, `projects`, `rabVersions`).
- **Cloud Storage**: Implementing real file uploads instead of local dummy files.

### 5. Document Intelligence
- Introducing a dedicated Python pipeline for PDF splitting, OCR, and Vision analysis using Gemini 1.5 Pro.

## What is Preserved?

- **The Deterministic Philosophy**: The core ADR (ADR 0002) dictating that AI does NOT do math remains the foundational principle.
- **Existing RAB Concepts**: The hierarchical structure of Divisions -> Items -> Resources established in v0.2 is preserved and formalized into shared schemas.
- **Legacy Reference**: `legacy/vite-v0.2/` remains in the tree as a historical reference.

## Rollout Strategy
1. Initialize monorepo scaffolding.
2. Port UI components to Next.js.
3. Build FastAPI Core Engine and define Pydantic schemas.
4. Implement Genkit orchestrator.
5. Wire up frontend to new backend APIs.
6. QA and release v0.3.0.
