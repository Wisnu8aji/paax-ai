# 📖 PAAX — GLOSSARY (istilah domain, 1 baris)

> Supaya AI tidak salah tafsir istilah teknik sipil / RAB Indonesia.
> Rumus kanonik lengkap: `CLAUDE.md` §5.

- **AHSP** — Analisa Harga Satuan Pekerjaan; sumber **koefisien** (Permen PUPR 8/2023). Bukan template output.
- **HSP** — Harga Satuan Pekerjaan = (A bahan + B upah + C alat) × (1 + BUK).
- **RAB** — Rencana Anggaran Biaya = Σ(Volume × HSP) + PPN.
- **BoQ** — Bill of Quantity; daftar volume pekerjaan.
- **BUK** — Biaya Umum & Keuntungan (overhead + profit); maks. 15% per Permen.
- **PPN** — Pajak Pertambahan Nilai (mis. 11%).
- **OH** — Orang-Hari (man-day); satuan koefisien tenaga kerja.
- **Volume** — kuantitas pekerjaan (m², m³, kg, ls, …).
- **Bobot item** — (Harga item / RAB total) × 100%.
- **Kurva S** — kumulatif progres rencana per periode (mingguan, dst).
- **CPM** — Critical Path Method: ES/EF (earliest), LS/LF (latest), float, jalur kritis.
- **Float (total)** — kelonggaran waktu tugas = LS − ES (0 = kritis).
- **Mandays** — Volume × Σ koef upah (OH); durasi = mandays ÷ pekerja efektif.
- **Skenario what-if** — frontier waktu–biaya, deterministik dari engine (bukan tebakan AI).
- **WBS** — Work Breakdown Structure; seksi pekerjaan (I..VII) untuk `/rab/build`.
- **HSPK** — Harga Satuan Pokok Kegiatan (varian harga regional).
